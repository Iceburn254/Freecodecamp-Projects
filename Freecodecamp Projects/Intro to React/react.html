Introduction to React

React is a JavaScript library developed by Facebook for building user interfaces, especially for single-page applications.

It allows developers to create large web applications that can update and render efficiently.

  • Component-based
  • Declarative syntax
  • Learn once, write anywhere
const element = <h1>Hello, world!</h1>;

JSX and Components

JSX is a syntax extension that looks similar to HTML and is used with React.

Components are the building blocks of React apps.

function Welcome() { return <h1>Hello, User!</h1>; }
  • Functional components
  • Class components
  • Reusability
State and Props

Props are used to pass data from parent to child components.

State is used to manage dynamic data within a component.

const [count, setCount] = useState(0);

You can pass props like this:

<Welcome name="Sam" />
Using Hooks

Hooks let you use state and other React features without writing a class.

  • useState
  • useEffect
  • useContext

The useEffect hook runs side effects in function components:

useEffect(() => {
console.log("Component mounted");
}, []);
